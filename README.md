php-fpm-omnibus
===============

PHP 5.6 with a set of commonly used modules.

Inherited from php-fpm
----------------------
* Composer
* PHPUnit
* PEAR
* libsodium
* zeromq

Included Modules
----------------
* curl
* gd
* mcrypt
* mongo
* mysqlnd
* pgsql
* redis

Preconfigured to listen on ```tcp/9000```.
